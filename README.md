# 🏠 家庭财务规划可调参模型

一个纯前端的交互式家庭财务规划工具，支持退休规划、教育金计算、资产增长模拟。所有计算在浏览器本地完成，无需后端服务。

**在线演示**：[https://hushing.github.io/family-finance-planner/](https://hushing.github.io/family-finance-planner/)

## 功能特性

- **退休资产预测** — 基于分段收益率，逐年计算资产增长曲线
- **通胀折算** — 绿色"购买力"列按今日币值折算，对抗通胀幻觉
- **熊市模拟** — 可指定某年触发暴跌，测试极端情景下的抗风险能力
- **教育金专户** — 独立计算子女教育基金（女儿/儿子分开账户）
- **安全提取率** — 自动计算退休提取率，≤4% 安全、4-5% 可接受、>5% 需警惕
- **实时交互** — 修改任意参数后图表和表格即时更新

## 可调参数

| 类别 | 参数 |
|------|------|
| 基础参数 | 当前年龄、退休年龄、规划终点年龄、初始资产、年投入、退休后年提取 |
| 分段收益率 | 36-45岁（激进期）、46-55岁（平衡期）、56-65岁（稳健期）、66岁+（保守期） |
| 养老年金 | 领取起始年龄、每年领取金额 |
| 宏观模拟 | 通胀率、熊市触发年份、熊市跌幅 |
| 教育金 | 年投入金额、预期收益率 |

## 技术栈

- 纯 HTML + CSS + JavaScript，单文件，零依赖构建
- [Chart.js](https://www.chartjs.org/)（CDN 加载）用于图表渲染
- 部署于 GitHub Pages

## 本地运行

直接用浏览器打开 `index.html` 即可，或启动本地服务器：

```bash
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 更新部署

修改 `index.html` 后执行：

```bash
git add . && git commit -m "update" && git push
```

GitHub Pages 会在约 1 分钟内自动重新部署。
